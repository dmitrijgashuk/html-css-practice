# HTML & CSS Practice

This repository contains a set of exercises for learning and improving HTML and CSS skills.

## Tasks

1. **task-01-basic-layout** — simple HTML structure with headers, paragraphs and images.
2. **task-02-flexbox-card** — a responsive card layout using Flexbox.
3. **task-03-navbar** — building a navigation bar with hover effects.

## How to run

Just open any `index.html` file in a browser or use Live Server in VS Code.
